# vertx-microservices-examples
Example code of using Vert.x for create Microservices blueprint 

## Compile:

mvn clean compile

## Test:

mvn clean test

## package

mvn clean package

## run vertical

java -jar microservices-vertx-1.0-SNAPSHOT-fat.jar
